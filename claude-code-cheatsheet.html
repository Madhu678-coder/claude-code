🪝
Hooks Lifecycle
1
PreToolUse Hook — The Security Gate
Blocks dangerous actions before execution
↓
2
Tool Executes
Bash command, file edit, or API call runs
↓
3
PostToolUse Hook — The Formatter
Auto-format code, clean output
Example: Prettier Hook
{
  "hooks": {
    "PostToolUse": [{
      "matcher": "Edit|Write",
      "command": "npx prettier --write $CLAUDE_FILE_PATH"
    }]
  }
}
Exit codes: 0 = success, 1 = error (continues), 2 = BLOCKS execution

CLI Power Moves
claude -p "prompt" One-shot mode (non-interactive)
claude -c Continue last session
claude --resume Resume by ID or name
claude -n "name" Name new session
claude --worktree Run in isolated git worktree
claude --effort high Max reasoning effort
claude --max-turns 5 Limit loop iterations
claude --max-budget-usd 5 Spending cap for session
